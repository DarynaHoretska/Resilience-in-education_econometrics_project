# Resilience of Secondary Education in Ukraine During Wartime

A quantitative analysis of how full-scale war (2022–2025) has affected
academic performance and student mobility in Ukrainian secondary education,
using individual-level National Multisubject Test (NMT) microdata and
regional administrative statistics from Derzhstat.

## Research Questions

- Does frontline proximity reduce NMT mathematics performance?
- Are students in conflict-affected regions more likely to sit the exam
  elsewhere (educational displacement)?
- Which factors sustain academic outcomes under wartime conditions
  (educational resilience)?

### Note on Git LFS

This repository uses [Git Large File Storage](https://git-lfs.com)
for CSV files in the `data/` directory. If you clone the repo and
intend to push large files, ensure LFS is installed on your machine:

```bash
# macOS
brew install git-lfs

# Ubuntu/Debian
sudo apt install git-lfs

# All platforms — run once after installation
git lfs install
```

Files tracked by LFS are defined in `.gitattributes`.
